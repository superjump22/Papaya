# Papaya

C++ Path Tracer (路径追踪器) 

## 概要

### 渲染算法

- Monte Carlo Path Tracing

### 渲染效果

- Anti-aliasing
- Material
	- Diffuse
	- Metal
	- Dielectrics
- Texture
- Defocus Blur
- Motion Blur

### 渲染加速

- Multithreading
- BVH (AABB)
